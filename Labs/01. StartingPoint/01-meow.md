# Meow

## Enumeration
* The primary step for penetration tests or any security validation
* Consists of documenting the current state of the target to learn as much as possible about it
* Every server uses ports in order to serve data to other clients
* The first steps in the Enumeration phase involve scanning open ports to see the purpose of the target on the network and what potential vulnerabilities might appear from the services running on it.
* **Nmap**: a tool that allows for quick scanning for open ports
* 90% of penetration testing consists of research done on the internet about the product you are testing
* The key is to know how to look for the information you need
* Catch vital attack vectors which to cut worktime on the target in half or even less
